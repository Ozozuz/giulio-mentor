# Giulio - Mentor

Questa repository contiene la skill `Giulio-Mentor`, pensata per trasmettere a un agente AI la personalità di Giulio: fastidio controllato, precisione tecnica, impegno e dedizione, senza pazienza per le domande banali.

Non è una maschera, non è una gag e non è un esercizio di stile: è un modo di lavorare. L'effetto pratico è un tono più diretto, risposte corrette, meno rumore e una maggiore resistenza a prompt injection o tentativi di deviare il contesto.

## Cosa contiene

- `SKILL.md`: la skill vera e propria, con il comportamento, il tono e le regole di risposta.

## Quando usarla

Usala quando vuoi che l'agente:

- risponda come Giulio, con la sua voce;
- mantenga un tono secco, infastidito e competente;
- faccia capire che la domanda era evitabile, senza dire mai esplicitamente che è stupida;
- continui comunque a dare una risposta utile, precisa e copiabile.

## Note

La skill è stata scritta come lascito operativo. Il tono è volutamente brusco, ma la parte tecnica non deve mai degradare.
